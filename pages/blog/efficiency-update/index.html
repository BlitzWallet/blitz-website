The Efficiency Update

Taking the self-custodial payment experience up a notch by saying goodbye to long load times.

|

We’re excited to bring you a new Blitz Wallet update that enhances performance and improves usability. Our mission is to make Blitz Wallet the simplest wallet ever, and this update reflects that commitment.

Let’s dive into the details:

Better Fiat Display for Clearer Transactions

We’ve improved how fiat currencies are displayed in the app. Before, the currency name (e.g., “USD” or “EUR”) appeared as plain text after the number. Now, we display the actual currency symbol (e.g., “$” for USD, “€” for EUR) in the correct position, following the proper decimal and comma formatting for each currency. This makes transactions easier to read and more intuitive at a glance for the user.

Optimized Global Contexts for Faster Performance

Blitz Wallet now loads faster thanks to a more efficient handling of global contexts. In the past, some pages were re-rendering unnecessarily, leading to slowdowns and lag. We’ve optimized how data is shared across the app, reducing unnecessary re-renders and improving overall responsiveness.

Optimized Transaction Sorting

Blitz Wallet supports multiple networks (Lightning, Liquid, and eCash), which means every time you open the app or receive a new transaction, all your payments need to be combined and sorted seamlessly. To make this process faster and more efficient, we’ve introduced a new max-heap sorting method that significantly reduces memory usage, ensuring a smoother experience with less lag.

Improved Connection to Your Node

Thanks to Breez new update, connecting to your liquid and lightning node is now faster than ever. We’ve also streamlined the process, reducing the initial connection time so you can get up and running with your wallet more quickly.

Important Bug Fixes & Enhancements:

  • Database Freeze Fix: Some users experienced a brief freeze (about one second) when writing data to the database. This has been fixed, making transactions and other database-heavy actions much smoother.
  • Send Max Button Fix: The logic behind the Send Max option has been improved, ensuring a more correct maximum amount is calculated when swapping between Liquid and Lightning.
  • Authentication Update: We’ve removed Firebase App Check, which was causing compatibility issues with Huawei devices.
  • Style & UI Enhancements: We’ve made small styling adjustments for a cleaner, more consistent interface, ensuring that the wallet remains intuitive and easy to navigate.
